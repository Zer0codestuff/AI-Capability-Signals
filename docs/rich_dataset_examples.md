# Rich Dataset Examples

Generated by:

```bash
uv run python -m frontier_ai.dataset_factory
```

Useful entry points:

- `data/dataset/dataset_manifest.csv`: table list, row counts, source IDs.
- `data/dataset/unified_model_index.parquet`: broad model index spanning Hugging Face, OpenRouter, Epoch AI and Open LLM Leaderboard rows.
- `data/dataset/lmarena_full.parquet`: full selected LMArena snapshots, not just latest top rows.
- `data/dataset/openllm_leaderboard_metrics_long.parquet`: long-format benchmark metrics from Open LLM Leaderboard JSON files.
- `data/dataset/huggingface_model_files.parquet`: repository file/weight/config metadata for top downloaded HF models.
- `data/dataset/swebench_instance_outcomes.parquet`: submission-by-instance SWE-bench Verified matrix.

Example:

```python
from pathlib import Path
import pandas as pd

root = Path("data/dataset")
manifest = pd.read_csv(root / "dataset_manifest.csv")
models = pd.read_parquet(root / "unified_model_index.parquet")
hf_files = pd.read_parquet(root / "huggingface_model_files.parquet")
openllm = pd.read_parquet(root / "openllm_leaderboard_metrics_long.parquet")

print(manifest.sort_values("rows", ascending=False)[["table", "rows", "columns"]])
print(models["source_namespace"].value_counts())
print(hf_files.groupby("extension")["size_bytes"].sum().sort_values(ascending=False).head())
print(openllm.groupby(["benchmark", "metric"])["value"].mean().sort_values(ascending=False).head(20))
```
